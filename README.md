# Weather Forecast Application

### Description

A simple web application that allows users to search for any city worldwide and view real-time weather conditions along with a seven-day forecast.

---

## Technologies

JavaScript, HTML, CSS, Public Weather API

## Duration

3 days

---

## Project Overview

This project is a web-based weather forecast application that enables users to search for any city and instantly view current weather conditions and a seven-day forecast. The application retrieves live weather data from a public API and presents it in a clear, simple, and user-friendly interface.

---

## Features

Search for weather information by city name
Display real-time weather data
Show a seven-day weather forecast
Responsive and easy-to-use interface
Basic error handling for invalid searches

---

## What I Implemented

API integration to fetch real-time weather data
Asynchronous JavaScript for handling API requests
User input handling and validation
Responsive interface design for better user experience

---

## What I Learned

API integration and working with external data sources
Asynchronous programming in JavaScript
Improving user experience through clear data presentation
